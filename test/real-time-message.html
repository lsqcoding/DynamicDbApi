消息API参考
1. 发送消息 (REST API)
POST /Message/send
请求体:
{ "topic": "general", "content": "消息内容", "type": "text", "target": "user123", // 可选，留空表示广播 "persist": true, // 是否持久化到数据库 "actions": [ // 可选，用于操作类型消息 { "label": "确认", "value": "confirm", "type": "primary" } ], "metadata": {} // 可选，附加元数据 }
响应:
{ "success": true, "messageId": "msg-12345", "timestamp": "2023-07-15T12:00:00Z" }
2. 获取消息历史
GET /Message/history
参数:
  • topic: 消息主题（可选）
  • limit: 消息数量限制（默认100）
  • offset: 偏移量（默认0）
  • startTime: 开始时间（可选）
  • endTime: 结束时间（可选）
响应:
{ "messages": [ { "id": "msg-12345", "topic": "general", "content": "消息内容", "type": "text", "sender": "user456", "timestamp": "2023-07-15T12:00:00Z", "metadata": {} } ], "total": 150, "page": 1, "pageSize": 100 }
3. WebSocket消息格式
连接时发送:
{ "type": "connect", "userId": "user123", "token": "your-auth-token", // 可选 "topics": ["general", "updates"] }
接收消息格式:
{ "type": "message", "data": { "id": "msg-12345", "topic": "general", "content": "消息内容", "type": "text", "sender": "user456", "timestamp": "2023-07-15T12:00:00Z", "actions": [], "metadata": {} } }
